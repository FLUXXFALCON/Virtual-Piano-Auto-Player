## Reporting a Vulnerability

We take the security of our users seriously. If you find a security vulnerability, please do not disclose it publicly through GitHub issues. Instead, please follow these steps:

1. **Email us**: Send a detailed report to the maintainer (FLUXXFALCON) or via GitHub's private vulnerability reporting feature.
2. **Details**: Include a description of the issue, steps to reproduce, and any potential impact.
3. **Response**: You will receive an acknowledgment within 48 hours.

We appreciate your help in keeping this tool safe for everyone.
